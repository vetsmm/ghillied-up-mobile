# Ghillied Up Mobile

### Development

Two environment variables are required to run. These should be pulled dynamically in the future
```
IDME_CLIENT_ID
IDME_CLIENT_SECRET
```

Run metro
```bash
yarn run start
```

### Optimize Assets before Upload
```bash
npx expo optmize .
```
